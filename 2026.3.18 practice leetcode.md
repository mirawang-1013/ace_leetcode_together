## 练习 2：什么能当 dict 的 key？

以下哪些可以当 dict 的 key？写出 ✅ 或 ❌，并说明原因：

```python
d = {}
d["abc"] = 1
d[[1, 2, 3]] = 2
d[(1, 2, 3)] = 3
d[{1, 2}] = 4
d[42] = 5
d[('a', [1, 2])] = 6
```

<details>
<summary>答案</summary>

```python
d["abc"] = 1            # ✅ string 不可变
d[[1, 2, 3]] = 2        # ❌ list 可变，unhashable
d[(1, 2, 3)] = 3        # ✅ tuple 不可变
d[{1, 2}] = 4           # ❌ set 可变，unhashable
d[42] = 5               # ✅ int 不可变
d[('a', [1, 2])] = 6    # ❌ tuple 里包含 list，整体不可 hash
```

规则：**key 必须是不可变的（hashable）**。即使是 tuple，如果里面嵌套了可变对象，也不行。
</details>

## 练习 3：遍历 dict

不运行代码，写出以下每段的输出：

**A:**
```python
count = {"apple": 3, "banana": 1, "cherry": 2}
for x in count:
    print(x)
```

**B:**
```python
count = {"apple": 3, "banana": 1, "cherry": 2}
for k, v in count.items():
    print(k, v)
```

**C（找错）：**
```python
from collections import Counter
nums = [1, 1, 2, 3, 3, 3]
count = Counter(nums)
for freq, num in count:
    print(freq, num)
```

<details>
<summary>答案</summary>

**A:** 只遍历 key
```
apple
banana
cherry
```

**B:** 遍历 key-value 对
```
apple 3
banana 1
cherry 2
```

**C:** 报错！`for freq, num in count` 只遍历 key（int），不能 unpack 成两个变量。

修正：
```python
for num, freq in count.items():  # 注意顺序：key 是数字，value 是频率
    print(num, freq)
# 1 2
# 2 1
# 3 3
```
</details>

## 练习 4：Bucket Sort 填空

补全以下代码，返回数组中出现频率前 k 高的元素：

```python
        for ___, ___ in ___:                # 填空 3, 4, 5
            bucket[___].append(___)         # 填空 6, 7

        result = []
        for i in range(___, ___, ___):      # 填空 8, 9, 10
            for ___ in ___:                 # 填空 11, 12
                result.append(___)          # 填空 13

```

<details>
<summary>答案</summary>

```python
from collections import Counter
from typing import List

class Solution:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        count = Counter(nums)
        bucket = [[] for _ in range(len(nums) + 1)]       # 1: []  2: len(nums) + 1

        for num, freq in count.items():                    # 3: num  4: freq  5: count.items()
            bucket[freq].append(num)                       # 6: freq  7: num

        result = []
        for i in range(len(bucket) - 1, 0, -1):           # 8: len(bucket)-1  9: 0  10: -1
            for num in bucket[i]:                          # 11: num  12: bucket[i]
                result.append(num)                         # 13: num
                if len(result) == k:
                    return result
```

关键点：
- 填空 1：`[]` 不是 `0`，因为每个位置要装多个数字
- 填空 2：`len(nums) + 1`，因为最大频率可能是 `len(nums)`
- 填空 3-5：`num, freq` 顺序不能反，且必须用 `.items()`
- 填空 6-7：`freq` 当 index，`num` 当 value（和直觉相反！）
- 填空 8-10：倒着走，从最高频率开始
- 填空 13：append 的是 `num` 不是 `i`
</details>

## 今日笔记总结

| 薄弱点 | 核心规则 | 记忆方法 |
|---|---|---|
| 类型转换 | `''.join()` 拼字符串，`list()` 转 list，各做各的 | 它们不是互逆操作 |
| dict key | 可变的不能当 key（list, dict, set） | 能变的就不可靠，不能当钥匙 |
| 遍历 dict | `for x in dict` 只拿 key，要 `.items()` 才有 value | 忘了 `.items()` = 忘了拿 value |
| index vs value | bucket sort 里 index 是频率，value 是数字 | 和平时反过来的，多 print 检查 |
